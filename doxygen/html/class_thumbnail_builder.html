Detailed Description

Scala listę obrazów w siatkę miniatur (mozaikę).

Miniatury mają zachowane proporcje wysokość/szerokość. Brakujące komórki siatki wypełniane są czernią.

Przykład użycia:

ThumbnailBuilder tb(128, 4);
cv::Mat mosaic = tb.build(imagePaths);
cv::imwrite("mosaic.png", mosaic);
