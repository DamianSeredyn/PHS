Detailed Description

Pula wątków producent–konsument.

Zadania (std::function<void()>) są kolejkowane i wykonywane przez stałą liczbę wątków roboczych. Destruktor czeka na opróżnienie kolejki.

Przykład użycia:

ThreadPool pool(4);
for (auto& path : paths)
pool.submit([&]{ processor.processImage(path, dst); });
pool.waitAll();
